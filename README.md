# CodeNinjaHub

A SEO-optimized programming languages website built with Next.js, featuring comprehensive guides for Python, Java, and C++.

## Features

- **SEO Optimized**: JSON-LD schemas, OpenGraph metadata, sitemap, robots.txt
- **Server-Side Rendering**: Fast page loads and better SEO
- **Responsive Design**: Mobile-friendly UI with Tailwind CSS
- **Modern Stack**: Next.js 14, React 18, shadcn/ui components

## Pages

- `/` - Homepage with language overview
- `/languages/python` - Python programming guide
- `/languages/java` - Java programming guide  
- `/languages/cpp` - C++ programming guide

## Tech Stack

- Next.js 14
- React 18
- Tailwind CSS
- shadcn/ui
- Lucide Icons

## Getting Started

```bash
# Install dependencies
yarn install

# Run development server
yarn dev

# Build for production
yarn build

# Start production server
yarn start
```
